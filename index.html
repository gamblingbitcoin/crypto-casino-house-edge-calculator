📐 How House Edge Is Calculated

The house edge represents the mathematical advantage the casino has on every bet you place. It's expressed as a percentage of each wager that the casino expects to keep over time.

For dice games:

// Fair payout = 1 / (win chance / 100)
// House edge = (1 - (actual payout / fair payout)) × 100

Win chance: 50% → Fair payout: 2.00×
Casino offers: 1.98× → House edge: 1.00%

For crash / limbo:

// The commission is taken from the prize pool each round
// House edge ≈ commission percentage

Commission: 1% → House edge: 1.00%

For roulette:

// House edge = number of zeros / total slots × 100

European: 1/37 = 2.70%
American: 2/38 = 5.26%

Return to Player (RTP) is simply 100% - house edge. An RTP of 99% means that for every $100 wagered, you can expect to get $99 back on average over a large number of bets.

Keep in mind that house edge describes the long-term mathematical expectation. Short-term results can vary wildly due to variance, which is why gambling should always be approached responsibly.
